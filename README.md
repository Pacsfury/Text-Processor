# Text Processor
Just a very basic text processor. At the moment, it has **2** main parts:

## Editor
In editor, you edit your files, using a toolbar including:
* Home to go to the menu
* An input to change file's name
* Bold, Italic and Underline
* Change font, using installed ones
* Change color to an HEX code
All the documents get saved.
## Menu
In menu, you can find all your files in a folder. You can see 2 buttons there:
* **Select Folder**: Used for selecting the folder where your files will be stored and being used
* **New Document**: Creates a new document in the selected folder
